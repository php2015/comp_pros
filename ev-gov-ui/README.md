### 项目安装

- 克隆
```bash
git clone https://gitee.com/yongchao9/sally-ui.git
```

- 安装依赖
```
npm install
```

- 启动项目
```
npm run serve
```

- 项目打包
```
np, run build
```
> 注:运行完成后，将dist文件夹中的内容放入对应的后台项目的static包下即可。

- Lints and fixes files
```
yarn run lint
```
